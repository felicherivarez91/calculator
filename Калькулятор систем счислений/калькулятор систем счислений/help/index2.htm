Основные элементы программы

Данная программа содержит следующие елементы: главное меню, поле для ввода значений,кнопки базовых арифметических операций, а также расширенные настройки.

Пункты для перевода чисел из различных систем(двоичная,восьмиричная,десятичная,шестнадцатеричная),а также пункты для перевода двоичного кода в следующие форматы: прямой код,обратный код,дополнительный код.
